**siCare** (Sistem Informasi Catatan, Presensi, Rekap, & Evaluasi) adalah aplikasi berbasis web yang dirancang khusus untuk mempermudah manajemen operasional internal perusahaan. Aplikasi ini mengintegrasikan pendataan karyawan (*employee profiling*), pencatatan kehadiran (*attendance tracking*), manajemen cuti, hingga otomatisasi kalkulasi penggajian (*payroll automation*) ke dalam satu platform yang terpusat dan aman.

Aplikasi ini dikembangkan dengan fokus pada kemudahan penggunaan (UX), efisiensi performa, dan akurasi data finansial, menjadikannya kelanjutan evolusi yang ideal dari sistem *gatekeeping* data sebelumnya.

---

## 🚀 Fitur Utama (Key Features)

- **Employee Data Management (Pendataan Karyawan):** Penyimpanan profil lengkap karyawan, jabatan, divisi, kontrak kerja, dan manajemen hak akses (Role-Based Access Control).
- **Attendance & Leave System (Presensi & Cuti):** Pencatatan kehadiran harian secara real-time, log keterlambatan, serta pengajuan cuti/izin digital yang terintegrasi.
- **Automated Payroll System (Penggajian Otomatis):** Kalkulasi gaji otomatis berdasarkan rekapitulasi presensi, tunjangan, potongan harian/bulanan, dan pembuatan slip gaji digital.
- **Reporting & Evaluation (Rekap & Evaluasi):** Dasbor analitik untuk melihat performa kehadiran karyawan dan laporan pengeluaran gaji bulanan perusahaan.

## 🛠️ Teknologi yang Digunakan (Tech Stack)

Aplikasi ini dibangun menggunakan ekosistem teknologi modern yang andal dan mudah di-scale:

- **Backend:** PHP (Laravel / Native) / Node.js *(Sesuaikan dengan stack Anda)*
- **Frontend:** HTML5, CSS3, JavaScript, TailwindCSS / Bootstrap
- **Database:** MySQL / PostgreSQL
- **Authentication:** Secure Session / Token-Based Authentication
